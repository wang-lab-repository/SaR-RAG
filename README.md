# SaR-RAG
SaR-RAG (Selective &amp; Refined RAG), a novel framework integrating two key components. The framework consists of five collaborative components: the Retrieval Decision Gate (RDG), the Hybrid Lexical-Semantic Retriever (HLSR), the Semantic-Aware Chunker (SAC), the Multi-Dimensional Document Scorer (MDDS), and the Context Aggregation Module (CAM).
